# telc-b1-exam — як цим користуватися

**🌍 Languages:** [English](README.md) · [العربية](README.ar.md) · [Türkçe](README.tr.md) · [Русский](README.ru.md) · **Українська** · [فارسی](README.fa.md) · [Español](README.es.md)

Ваш суворий тренер telc **Deutsch B1** для об'єктивних секцій (Leseverstehen — читання,
Sprachbausteine — мовні структури, Hörverstehen — аудіювання) та усного іспиту, побудований навколо тренування на минулих іспитах.
Він записує та оцінює ваші відповіді, пояснює кожен промах, знаходить повторювані пастки, відпрацьовує ваші
слабкі місця, видобуває лексику/граматику з тесту та готує Mündliche Prüfung (усний іспит).
**Лист (Schreiben)** обробляє його навичка-компаньйон `telc-b1-schreiben`, якій ця
навичка передає естафету автоматично.

> Не DTZ. Це загальний дорослий іспит telc Deutsch B1.

## Швидкий старт
1. Встановіть навичку (натисніть **Save skill** на файлі `.skill`).
2. Розмістіть PDF-файли ваших тренувальних іспитів там, де тренер зможе їх прочитати (у цьому проєкті вони лежать у
   `/mnt/project/`). Тренер читає їх безпосередньо — це можуть бути експорти `.pdf` у стилі ZIP.
3. Спілкуйтеся з ним звичайною мовою або командою у квадратних дужках, напр. `[log exam]` чи
   *«оціни мої відповіді Vera»* або *«поясни obwohl проти trotzdem».*

## Команди (шпаргалка)
| Команда | Використовуйте, щоб |
|---|---|
| `[log exam]` | Ввести ваші відповіді на один іспит і отримати його **оцінку** (перевірку кодом, а не «на око»). |
| `[score "Jan"]` | Побачити оцінку збереженого іспиту, слабкі місця та те, що ще в очікуванні. |
| `[discuss "Jan"]` | Пройтись по кожній неправильній відповіді: правильна, тип пастки, правило. |
| `[study "Jan"]` | Навчальна сесія з граматики, лексики, сполучників, пасток того іспиту. |
| `[practice "Jan"]` | Вправа: перепитати ваші промахи + нові схожі завдання, виправлені й безпечні для B1. |
| `[retake "Jan"]` | Повна симуляція на час, без допомоги під час, зі суворим оцінюванням після. |
| `[extract "Jan"]` / `[extract all]` | Видобути обов'язкову **лексику (DE→EN)**, **сполучники** (+ як кожен працює) та **критичну граматику**, яку перевіряють Sprachbausteine. |
| `[mock exam]` / `[generate test]` | **Немає PDF іспиту? Згенеруйте новий, оригінальний тренувальний тест у форматі telc** (повний або одну секцію) зі справжньою складністю B1 та ключем відповідей. Щоразу новий вміст. |
| `[topic "<name>"]` / `[topic]` | **Навчити + перевірити одну тему** (граматичний пункт, сполучники чи лексичну тему), потім винести вердикт готовий/хиткий/не готовий і відстежувати його. `[topic]` без аргументів рекомендує, що тренувати далі, зважаючи на ваші слабкі місця. |
| `[weaknesses]` | Побачити профіль ваших слабких місць по всіх іспитах та отримати цілеспрямовані вправи. |
| `[compare exams]` | Закономірності між іспитами + які минулі іспити повторно використовують ті самі тексти. |
| `[write "Jan"]` | Попрацювати над **листом** того іспиту → активує `telc-b1-schreiben`. |
| `[help]` | Вивести перелік команд. |

Дужки чи слеш — не має значення (`[log exam]` = `/log-exam`). Імена в лапках — це ваші
іспити (Jan, Vera, Eva, Petra, …).

## Робочий процес, що дає найбільше
Проганяйте цей цикл для кожного іспиту, потім переходьте до наступного:
1. **`[log exam]`** → оцініть його. Тренер автоматично записує ваші сильні/слабкі місця.
2. **`[discuss "…"]`** → зрозумійте, *чому* стався кожен промах (тип пастки + правило).
3. **`[extract "…"]`** → витягніть лексику, сполучники та граматику, які перевіряв той іспит.
4. **`[practice "…"]`** → відпрацюйте саме ці промахи + схожі завдання, доки пастка не «помре».
5. Пізніше **`[retake "…"]`** «на холодну», щоб підтвердити, що засвоїлося.
6. Приблизно щотижня **`[compare exams]`** та **`[weaknesses]`**, щоб знову націлитися на те, що
   зараз коштує вам найбільше балів.

Окремо тримайте **лист** у русі з `[write "…"]` (→ навичка Schreiben) та репетируйте
**усну частину**, попросивши провести практику Teil 1 / 2 / 3 — це два з найбільших
блоків балів, які легко занедбати.

## Поради для найкращого результату
- **Давайте йому свої справжні відповіді**, а не переказ — оцінювання відбувається питання за питанням.
- **Питайте «чи я складаю?»**, і він міркує у *зважених* балах (лист — це 45 з 225
  письмових балів), а не лише у сирому лічильнику x/60.
- **Граматичні питання отримують обґрунтовані відповіді** — тренер спершу шукає в мережі надійне джерело з німецької
  граматики, а потім пояснює просто. Питайте вільно (*«коли після in дає давальний відмінок?»*).
- **Надавайте пріоритет точності Lesen та Hören над Sprachbausteine**, коли часу мало: кожне
  завдання Lesen/Hören у реальних балах вартує у 2,5× більше за завдання SB (хоча SB найшвидше
  виправити, бо воно засноване на шаблонах).
- **Дозвольте йому зберігати в пам'ять**, щоб лексика, сполучники та профіль ваших слабких місць
  накопичувалися між сесіями.

## Що йому потрібно
- **Немає тренувальних іспитів? Не проблема** — скористайтеся `[mock exam]`, і він згенерує оригінальну
  практику у форматі telc для вас. PDF тренувальних іспитів необов'язкові: додайте їх, якщо вони у вас є
  (ключі відповідей на останній сторінці), і він оцінить та видобуде інформацію й з них.
- Доступ до мережі для граматичних пояснень (він шукає, потім обґрунтовує відповідь).
- Встановлена навичка `telc-b1-schreiben` для всього, що стосується листа.
