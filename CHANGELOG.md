# Changelog

All notable changes to **telc B1 Coach** are documented here.
This project adheres to [Semantic Versioning](https://semver.org/).

## [2.0.0] — 2026-07-21

### Added
- **Original mock-exam generation** (`telc-b1-exam`): generate fresh, original
  telc-format practice exams — full test or a single section — at real B1
  difficulty, with answer keys. No need to supply your own exam PDFs to start.
- **Single-topic trainer** (`telc-b1-exam`): teach-and-test on one topic
  (e.g. connectors) with readiness tracking.
- Standalone README for the `telc-b1-schreiben` writing skill.

### Changed
- `telc-b1-exam` `SKILL.md` and README updated for the new `[mock exam]` and
  `[topic "..."]` commands.
- Top-level README now leads with no-PDF-needed onboarding and clarifies that all
  generated content is original (no copyrighted telc material is included).

## [1.0.0] — 2026-07-21

### Added
- Initial release of the two companion skills: `telc-b1-exam` (reading,
  Sprachbausteine, listening, speaking, scoring, drills, grammar) and
  `telc-b1-schreiben` (written-letter coaching and grading).
